    # 白嫖亚马逊云服务器

    今天来分享一下如何白嫖亚马逊云服务器

    国内服务器厂商BAT后两家的服务器，最便宜的也是500左右一年，3年5折900多块虽然便宜一点，但看起来也是一笔巨款花销。亚马逊云AWS为了吸引客户，免费1年体验一直就在那，看见AWS官网绚丽的免费宣传广告就下面图里面的蓝字，就问你心动不心动？

    # 前提

    没错，这个白嫖也是有前提条件的

    • 由于亚马逊云AWS是国外厂商，虽然也有国内站点，但是访问速度可能不是很理想。你可能需要科学上网才能快速访问。
    • 你得有一个亚马逊云的账号，不知道怎么注册的话google吧。
    • 体验过其他免费蛋糕的，不确定可以白嫖

    几个连接

    • 注册AWS
    • AWS控制台
    • AWS虚拟服务器列表

    # 创建虚拟机

    顺利得到账号后就可以开始创建服务器了

    # Step 1

    打开虚拟控制台，点击图中启动虚拟机，开始创建流程

    # Step 2

    下面就是创建页面的第一步

    • 在搜索框中输入centos
    • 点击社区AMI
    • 选择一个你中意的Cent OS版本
    • 点击下一步

    # Step 3

    • 这里要选一个免费的实例配置 下一步

    # Step 4

    无需多说，下一步

    # Step 5

    无需多说，下一步

    # Step 6

    无需多说，下一步

    # Step 7

    • 这里注意，如果没有你需要的端口要手动添加
    • 比如22端口，windows的话3389端口 这一步如果端口忘记设置的话，会连不上服务器。 设置好了就点击审核和启动

    # Step 8

    这里是创建的最后一步了，可以点击顶部的步骤返回去修改参数设置。确认无误的话，就启动吧。

    # Step 9

    这里会弹出一个设置密钥对的对话框，后面登录服务器就靠它了

    • 第一项下拉选择创建新的密钥对
    • 第二项输入便于识别的密钥对文件名称，特别是当你的下载目录文件很多的时候，方便找😂

    # Step 10

    到这里等待启动就行了，过程应该3到5分钟

    # 设置虚拟机

    点击Step 10图片中的服务器id，或者从上面提供的服务器列表连接进入你的服务器设置/详情页面，开始连接并设置服务器。

    # Step 1

    • 点击服务器概要标题右侧的连接按钮

    # Step 2

    连接页面，这里有几种连接方式，第三种是我熟悉的😂，前面2中还需要再研究

    • 点击下面的SSH客户端标签页
    • 复制第4步下面的示例命令

    # Step 3

    • 掏出你惯用的命令行工具，这里我用Git Bash演示。
    • 转到启动之前下载的pem文件所在的目录
    • 粘贴命令，回车

    登录过程中有一次询问 老师说过的，第一次登录会提示天价known hosts，选yes，回车

    # Step 4

    看到控制台在$后面闪烁的光标的话，Congratulations ！ 登录成功

    # Step 5

    每次登录都依赖于pem文件，对于学习来说，不免过于麻烦。 接下来开启密码登录

    # 下面这行命令是强制设置用户`centos`的密码
> sudo passwd centos
Changing password for user root.
New password:              #输入密码第一次
Retype new password:       #输入密码第二次
passwd: all authentication tokens updated successfully.

# 用vi编辑器修改登录配置文件，关键配置如下
# 使用↑↓←→导航光标
# 按 字母 i进入插入模式
# 按ESC，输入:wq，保存退出

> vi /etc/ssh/sshd_config
...
PasswordAuthentication yes #允许使用密码登陆，由no改成yes
PermitRootLogin yes        #如果需要使用root登陆，由no改成yes，如果不需要就不要动
... 

# 重启机器
> reboot
